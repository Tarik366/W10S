# W10S
Windows 10 Speech recognition fix

This repository for Windows 10 Speech recognition language bug and linux systems(Don't tested on linux)

# Installation
Just clone the repository with this command
```
git clone https://github.com/Tarik366/W10S.git
```
or just download source code from this [link](https://github.com/Tarik366/W10S/releases/latest).


Except starting with boot is same for all platforms you are only start the [KeyChecker](https://github.com/Tarik366/W10S/blob/main/KeyChecker.py) file. If you wanna start program with the boot, you need the follow this steps
## Windows
In folder manager, press with the right click to start.bat and create shortcut of file. And move the file to `C:\Users\%USERPROFILE%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup` folder.
